  1. 现金流质量优秀：每股经营现金流1.44元远超每股收益0.48元，证明盈利有真实现金流支撑
  2. 战略股东进场：5%以上股东完成过户，是真金白银的背书
  3. 募投项目结项：进入收获期，可能带来后续利润释放
  4. 赛道长期价值：清洁能源是未来十年的投资主线

保守分析师最强论点

  1. 静态PE被刻意回避：实际估值45.51倍而非宣称的34.49倍
  2. 剪刀差无法解释：5%营收增长如何支撑25.77%净利润增长？增长来源不透明
  3. 流动性指标恶化：流动比率和速动比率双低，短期偿债存疑
  4. 异常波动悖论：2月10号至今股价并未大幅上涨，"先知先觉"逻辑自相矛盾

中性分析师关键观察

  • 技术分析可靠性受限于数据不足
  • 激进派的"向下比较"逻辑有瑕疵
  • 保守派的"完全观望"忽视了时间成本
  • 建议在年报窗口期前保持灵活性

二、核心争议裁决

争议一：估值到底贵不贵？

裁决：保守分析师胜出

激进分析师全程使用34.49倍PE（TTM），刻意回避了45.51倍静态PE。这一数字游戏是最大的逻辑漏洞。

当前13.81元对应静态PE 45.51倍，营收增速仅5.07%。这个估值水平意味着市场对公司未来业绩有极高的预期溢价，一旦年报不及预期，向下调整空间巨大。

争议二：净利润剪刀差如何解释？

裁决：信息不足，无法判断，但风险归因于多头

激进分析师将25.77%净利润增长解释为"经营效率释放"，但未能提供具体驱动因素。保守分析师提出的三种可能性（费用压缩、基数效应、非经常性损益）中，任何一种都无法保证可持续性。

更关键的是：如果是费用压缩，能持续压多久？如果是基数效应，2026年一季度会不会"变脸"？如果是非经常性损益，那当前估值的基础就是虚假的。

在年报验证之前，这个剪刀差是风险项而非加分项。

争议三：异常波动说明了什么？

裁决：保守分析师更有说服力

激进分析师的"先知先觉资金布局"论存在致命缺陷：如果真是利好提前反应，股价从2月10号到现在应该有明显涨幅。但实际上，当前价格与异常波动前并无显著差异。

这说明两种可能：

  • 可能性A：资金知道的是中性或负面信息
  • 可能性B：异常波动涉及未公开的重大事项

无论哪种，对外部投资者都不友好。

指标说明

股价曲线 = 当天用于回放估值/执行的价格口径；当前日级回放默认使用 14:55 附近的可见价格或当日可用标记价格。
持仓金额曲线 = 日末持股数 × 当天股价。
真实总账户净值 = 真实账户现金 + 全部真实持仓市值。这是执行层真正发生买卖和记账的总账户口径。
真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。
策略拆分盈亏 = 策略覆盖资产 - 累计分配资金。
策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
