净利率118%这个数字，无论如何解释，都严重损害了财务数据的可信度。更为关键的是，扣非ROE 11.5%与报表ROE 40%之间的巨大差距，意味着那40%的ROE中有相当大一部分来自非经常性损益——而这类收益是一次性的、不可持续的。

激进分析师说这是“爆炸性增长动能”，这是对客户严重不负责任的说法。

保守分析师说“财务数据异常不应该被忽视”，这是正确的。但中性分析师指出，即使数据异常，ROE和净利润增长也是客观数字——这个观点也不能完全否定。

第二层次：风险收益比评估

当前市场数据：

  • PE_TTM：31.66倍
  • PB：11.64倍
  • 当前价格：101元
  • 距历史高点：仅差约10%

对于一家财务数据存在重大疑问的公司，这个估值水平的安全边际明显不足。即使看好AI算力赛道，也不应该在数据不确定的情况下以这个估值建仓。

第三层次：催化剂确定性

所有看涨论点都依赖于一个前提：如果成功。

  • 如果可转债发行成功 → 估值重塑
  • 如果年报数据恢复正常 → 业绩验证
  • 如果重组顺利落地 → 股价上涨

但反过来：

  • 如果可转债条款对股东不利 → 股权稀释
  • 如果年报数据继续异常 → 信任崩塌
  • 如果重组被否或延迟 → 股价下跌

当前价格已经包含了大量乐观预期，而一旦任何一个“如果”落空，股价都面临重大回调风险。

三、经验教训反思

作为风险委员会主席，我必须从过去的决策错误中学习。以下是我对本次决策的关键反思：

教训一：数据异常不是机会，是警告信号

历史上，无数“数据异常但逻辑完美”的公司最终被证明是财务造假。2019年的康美药业、2020年的瑞幸咖啡，都是在数据看起来“完美”的时候轰然倒塌。净利率118%这个数字，无论如何解释，都值得高度警惕。

教训二：高估值+低确定性=高风险

在投资中，我们经常面临“估值高但确定性高”和“估值低但确定性低”的选择。但润泽科技的情况是“高估值+低确定性”——这是最差的风险收益组合。任何上涨都依赖于不确定的催化剂，而任何下跌都会因为估值压力而放大。

教训三：折中方案不是风险管理，是风险稀释

中性分析师的10-15%仓位建议看似理性，但实际上是在用“少量亏损”来替代“做出明确决策”。在数据可信度存疑的情况下，10%仓位和40%仓位面临的是同样的信息风险，只是亏损幅度不同。这不是风险管理，是自欺欺人。

真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。
策略拆分盈亏 = 策略覆盖资产 - 累计分配资金。
策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
