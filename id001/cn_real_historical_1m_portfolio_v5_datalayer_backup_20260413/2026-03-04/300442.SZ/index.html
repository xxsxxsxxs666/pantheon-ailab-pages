从本次分析中学到的关键教训

教训一：扣非数据与名义数据的差距是分析中最容易被忽视的盲点。

激进分析师的论述提醒我：高增长数字令人印象深刻，容易让人忽视数据质量的深层问题。 润泽科技案例中，210%的净利润增长与11.51%的扣非ROE之间存在71%的差距，这个差距往往意味着公司依赖非经常性损益来支撑表面增长，而非可持续的主营业务改善。

未来行动： 在分析任何高增长公司时，必须同时审视扣非数据与名义数据的差距。若差距过大，必须追问原因，而非仅凭名义数字做出投资决策。

教训二：流动性指标的双重告警应比技术面下跌更早触发警示。

技术分析往往关注价格走势和成交量，但流动性指标反映的是公司财务健康的底层逻辑。本次分析中，流动比率和速动比率双低于1的信号被我和其他分析师放在次要位置，但这恰恰是公司财务风险的最直接体现。

未来行动： 在评估任何投资公司时，流动性指标应作为先于技术面的第一道筛选标准。若流动性告警，必须谨慎评估，即使其他指标看起来很乐观。

教训三：技术分析的“解读”存在巨大主观性，不能作为单一决策依据。

激进分析师将量比0.65解读为“最后一跌”，但中性分析师正确指出：这同样可以解读为买盘衰竭。在图形完全一致的情况下，不同分析师可以得出完全相反的结论。

未来行动： 技术分析应作为辅助工具，而非决策核心。单独依赖技术指标（尤其是单一指标）进行投资决策是危险的。技术信号必须与基本面分析相互验证。

教训四：“仓位管理”比“选边站队”更能应对不确定性。

本次辩论中，激进分析师建议“适度加仓”，保守分析师建议“不持有或小仓位试探”，两者都在试图给出“正确”的方向判断。但中性分析师提出了更合理的思路：承认不确定性，用仓位控制来应对。

未来行动： 在不确定性高的情境下，应采用仓位管理策略，而非试图预测“正确”的方向。即使对某只股票有强烈判断，也应设置明确的止损位和仓位上限。

激进分析师的论述有其价值，但存在三个根本性缺陷：

第一，混淆了“数据存在”与“数据健康”的概念。 激进分析师强调“扣非ROE 11.51%本身很健康”，这个论证逻辑是成立的，但他们随即跳跃到“210%增长因此可信”，这个推断跳跃过大。更致命的是，他们完全回避了**销售净利率118%**这个数字的商业合理性问题——无论怎么解释，卖100元赚118元在正常商业逻辑下都是不可持续的。

第二，流动性分析存在致命漏洞。 激进分析师将可转债类比“房贷申请”，认为当前流动性压力可以接受。但关键差异在于：房贷申请人至少已有收入证明和审批进行中，而润泽的可转债仍在预案阶段，距离正式发行还有证监会受理、反馈、审核、注册等多个环节。在漫长的等待期内，公司每天面对的是真实的债务到期压力，而非“未来会好的”乐观预期。

第三，技术分析过于主观。 激进分析师将量比0.65解读为“卖压衰竭”，但中性分析师的反驳切中要害：缩量下跌同样可以解读为买盘不积极。两种解读在图形上完全一致，在趋势确认前，任何一种都是一厢情愿的猜测。

教训一：扣非数据与名义数据的差距是分析中最容易被忽视的盲点。

激进分析师的论述提醒我：高增长数字令人印象深刻，容易让人忽视数据质量的深层问题。 润泽科技案例中，210%的净利润增长与11.51%的扣非ROE之间存在71%的差距，这个差距往往意味着公司依赖非经常性损益来支撑表面增长，而非可持续的主营业务改善。

未来行动： 在分析任何高增长公司时，必须同时审视扣非数据与名义数据的差距。若差距过大，必须追问原因，而非仅凭名义数字做出投资决策。

教训二：流动性指标的双重告警应比技术面下跌更早触发警示。

技术分析往往关注价格走势和成交量，但流动性指标反映的是公司财务健康的底层逻辑。本次分析中，流动比率和速动比率双低于1的信号被我和其他分析师放在次要位置，但这恰恰是公司财务风险的最直接体现。

未来行动： 在评估任何投资公司时，流动性指标应作为先于技术面的第一道筛选标准。若流动性告警，必须谨慎评估，即使其他指标看起来很乐观。

教训三：技术分析的“解读”存在巨大主观性，不能作为单一决策依据。

激进分析师将量比0.65解读为“最后一跌”，但中性分析师正确指出：这同样可以解读为买盘衰竭。在图形完全一致的情况下，不同分析师可以得出完全相反的结论。

未来行动： 技术分析应作为辅助工具，而非决策核心。单独依赖技术指标（尤其是单一指标）进行投资决策是危险的。技术信号必须与基本面分析相互验证。

教训四：“仓位管理”比“选边站队”更能应对不确定性。

本次辩论中，激进分析师建议“适度加仓”，保守分析师建议“不持有或小仓位试探”，两者都在试图给出“正确”的方向判断。但中性分析师提出了更合理的思路：承认不确定性，用仓位控制来应对。

未来行动： 在分析任何高增长公司时，必须同时审视扣非数据与名义数据的差距。若差距过大，必须追问原因，而非仅凭名义数字做出投资决策。

教训二：流动性指标的双重告警应比技术面下跌更早触发警示。

技术分析往往关注价格走势和成交量，但流动性指标反映的是公司财务健康的底层逻辑。本次分析中，流动比率和速动比率双低于1的信号被我和其他分析师放在次要位置，但这恰恰是公司财务风险的最直接体现。

未来行动： 在评估任何投资公司时，流动性指标应作为先于技术面的第一道筛选标准。若流动性告警，必须谨慎评估，即使其他指标看起来很乐观。

教训三：技术分析的“解读”存在巨大主观性，不能作为单一决策依据。

激进分析师将量比0.65解读为“最后一跌”，但中性分析师正确指出：这同样可以解读为买盘衰竭。在图形完全一致的情况下，不同分析师可以得出完全相反的结论。

未来行动： 技术分析应作为辅助工具，而非决策核心。单独依赖技术指标（尤其是单一指标）进行投资决策是危险的。技术信号必须与基本面分析相互验证。

教训四：“仓位管理”比“选边站队”更能应对不确定性。

本次辩论中，激进分析师建议“适度加仓”，保守分析师建议“不持有或小仓位试探”，两者都在试图给出“正确”的方向判断。但中性分析师提出了更合理的思路：承认不确定性，用仓位控制来应对。

未来行动： 在不确定性高的情境下，应采用仓位管理策略，而非试图预测“正确”的方向。即使对某只股票有强烈判断，也应设置明确的止损位和仓位上限。

股价曲线 = 当天用于回放估值/执行的价格口径；当前日级回放默认使用 14:55 附近的可见价格或当日可用标记价格。
持仓金额曲线 = 日末持股数 × 当天股价。
真实总账户净值 = 真实账户现金 + 全部真实持仓市值。这是执行层真正发生买卖和记账的总账户口径。
真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。
策略拆分盈亏 = 策略覆盖资产 - 累计分配资金。
策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
