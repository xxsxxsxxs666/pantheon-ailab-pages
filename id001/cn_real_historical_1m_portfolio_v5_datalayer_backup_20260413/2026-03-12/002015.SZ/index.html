市场的真理是：强势股不给舒服的买点。 试图等完美买点的人，往往最终在高位追入。

五、关键观察指标与应急预案

持仓期间必须跟踪的信号

信号一：临时股东会内容（3月中下旬）

  • 如果审议通过实质性转型方案（资产注入、新业务布局、引入战投）→ 加仓到60-70%
  • 如果只是常规融资议案 → 保持当前仓位，等待年报验证
  • 如果议案未通过或内容负面 → 减仓至20%，止损位上移至19.50元

信号二：担保公告的后续披露

  • 如果担保规模在10亿以内 → 风险可控，维持仓位
  • 如果担保规模超过30亿 → 减仓观望，重新评估流动性风险

信号三：成交量变化

  • 警戒线：成交量比低于1.2，或出现“价涨量缩”背离
  • 减仓信号：价格创新高但成交量持续萎缩
  • 健康信号：成交量比维持在1.5以上，量价配合

信号四：监管问询函

  • 如果交易所发出问询函 → 立即减仓30%，回避短期风险
  • 如果只是公司自查公告且结果正面 → 影响中性，维持仓位

错误二：将“公告密集”简单解读为利好

之前有公司发布类似公告组合但最终落空。这次我不会仅凭预期就重仓，而是在当前位置建立合理底仓，等待催化剂兑现后再加仓。

  • 仓位分批建：20%起步，回调加仓，不一把梭
  • 止损明确设：18.50元破位坚决离场，不侥幸
  • 催化剂等确认：临时股东会结果出来前，仓位不超过50%

真实总账户净值 = 真实账户现金 + 全部真实持仓市值。这是执行层真正发生买卖和记账的总账户口径。
真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。

累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。

策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。

未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
