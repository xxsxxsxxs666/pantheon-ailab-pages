  • PB 10倍意味着市场对公司未来增长给出了极高的预期溢价

  • 激进分析师将IDC类比“电力公司”，但这个故事需要业绩持续兑现来支撑，而当前的现金流数据恰恰在质疑这一点

三、对激进分析师论点的批驳

3.1 "6个月数据已过时"是逃避而非论证

激进分析师的核心逻辑是“扣非ROE是旧数据，6个月内可能已变化”。但这个论点的本质是用不确定性来回避质疑，而不是解决质疑。

正确的投资逻辑应该是：在获得新数据之前，基于现有最强证据做决策。我们不能因为“未来可能变好”就去忽略“当前明显的问题”。

3.2 管理层回购不是“信心”的代名词

激进分析师将回购解读为“管理层认为股价被低估”。但：

  • 回购动机是多元的：支撑股价配合资本运作、股权激励、税务优化等
  • 在可转债预案通过后回购，可能是为了稳定股价以利重组推进，而非单纯的“价值发现”
  • 我们不能单凭一个动作就断定管理层的真实意图

3.3 "先知先觉"的前提是概率优势

激进分析师引用“在别人恐惧时贪婪”，但这句话的前提是——你要有足够的证据证明价格已经低于价值。当前的情况是：

  • 盈利质量存疑（现金流缺口1.05元）
  • 估值处于高位（PB 10倍）
  • 趋势明确向下（均线空头排列）

在这种“看不清”而非“明显低估”的情况下，激进建仓不是在逆向投资，而是在用本金冒险。

3.4 盈亏比分析揭示激进策略的致命缺陷

股价曲线 = 当天用于回放估值/执行的价格口径；当前日级回放默认使用 14:55 附近的可见价格或当日可用标记价格。
持仓金额曲线 = 日末持股数 × 当天股价。
真实总账户净值 = 真实账户现金 + 全部真实持仓市值。这是执行层真正发生买卖和记账的总账户口径。
真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。
策略拆分盈亏 = 策略覆盖资产 - 累计分配资金。
策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
