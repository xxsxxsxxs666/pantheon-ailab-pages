持有策略的风险

  1. 踏空风险：如果市场快速拉升（如铜价大涨、收购细节披露），可能错失机会
  2. 时机风险：等待太久可能错过最佳建仓窗口
  3. 心理压力：持有现金等待需要对自身判断有信心

应对措施

  1. 设置"追涨触发价"：如果收盘价连续2日站稳52元以上，考虑右侧追入（仓位不超过3%）

八、特别提醒：被忽视的风险因素

汇率风险（中性分析师提出，委员会认为重要）

  • SolGold是海外公司，未来收益结算涉及人民币汇率波动
  • 人民币升值 → 海外资产换算价值缩水
  • 人民币贬值 → 海外资产换算价值增加，但美元计价的开发成本上升
  • 这个风险在短期持仓中可能不明显，但在6个月以上持仓中是重要波动来源

供给端风险（双方均忽视）

  • 保守分析师提到"供给端扩张"，但未深入分析
  • 秘鲁、刚果金、智利的新增铜矿产能正在密集投产
  • 新增铜矿从勘探到投产需要5-10年，但一旦投产就会对价格形成压制
  • 需求的爆发和供给的释放之间存在时间差，这个时间差是当前铜价的主要支撑，但不会永远持续

回顾类似案例，周期股投资中最常见的错误是：

  1. 在景气高点用PE判断估值便宜（保守分析师警告的周期陷阱）
  2. 把小反弹当成趋势反转（激进分析师的误判）
  3. 把海外收购当成确定性利好（忽视了整合风险和融资成本）
  4. 忽视财务数据的时效性（在强周期行业尤其致命）

委员会认为：当前江西铜业的处境与上述错误高度吻合。

四、最终裁决：持有（观望）

裁决依据

我选择持有，不是因为双方都有道理而骑墙，而是因为当前市场状态确实不支持明确的方向判断。

关键理由：

  1. 数据时效性问题不可忽视：5个半月的财务数据滞后，在大宗商品强周期行业里是致命的参考缺陷。铜价在这期间可能剧烈波动，但投资者完全蒙在鼓里。

  2. 技术面的弱势判断更加客观：量比0.79不能证明抛压衰竭，只能说明观望情绪浓厚。49元支撑尚未经过有效确认，51.77元压力位难以突破，技术面并未给出明确方向。

  3. 净利率极低意味着容错空间极小：1.64%的净利率意味着公司几乎没有犯错的余地。激进分析师只强调上涨杠杆，却回避下跌风险——这是不完整的风险评估。

  4. 收购价值存在重大不确定性：对价、融资、开发周期等关键数据均未披露，盲目乐观是不专业的表现。

  5. 中性分析师的方案过于模糊：10%仓位金字塔建仓缺乏明确的触发条件，在方向不明的情况下执行难度高。

七、风险提示

持有策略的风险

  1. 踏空风险：如果市场快速拉升（如铜价大涨、收购细节披露），可能错失机会
  2. 时机风险：等待太久可能错过最佳建仓窗口
  3. 心理压力：持有现金等待需要对自身判断有信心

应对措施

  1. 设置"追涨触发价"：如果收盘价连续2日站稳52元以上，考虑右侧追入（仓位不超过3%）
  2. 分批建仓：不要一次性全仓，留有余地
  3. 严格止损：任何情况下，单次亏损不超过总仓位的1.5%

八、特别提醒：被忽视的风险因素

汇率风险（中性分析师提出，委员会认为重要）

  • SolGold是海外公司，未来收益结算涉及人民币汇率波动
  • 人民币升值 → 海外资产换算价值缩水
  • 人民币贬值 → 海外资产换算价值增加，但美元计价的开发成本上升
  • 这个风险在短期持仓中可能不明显，但在6个月以上持仓中是重要波动来源

供给端风险（双方均忽视）

  • 保守分析师提到"供给端扩张"，但未深入分析
  • 秘鲁、刚果金、智利的新增铜矿产能正在密集投产
  • 新增铜矿从勘探到投产需要5-10年，但一旦投产就会对价格形成压制
  • 需求的爆发和供给的释放之间存在时间差，这个时间差是当前铜价的主要支撑，但不会永远持续

股价曲线 = 当天用于回放估值/执行的价格口径；当前日级回放默认使用 14:55 附近的可见价格或当日可用标记价格。
持仓金额曲线 = 日末持股数 × 当天股价。
真实总账户净值 = 真实账户现金 + 全部真实持仓市值。这是执行层真正发生买卖和记账的总账户口径。
真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。
策略拆分盈亏 = 策略覆盖资产 - 累计分配资金。
策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
