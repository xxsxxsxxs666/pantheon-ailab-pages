  • 评估正面催化剂的概率和幅度
  • 评估负面催化剂的概率和幅度
  • 计算期望值，而非只考虑最好情景

对于润泽科技：

  • 正面催化剂：可转债超预期、重组成功、盈利质量验证（概率中等）
  • 负面催化剂：可转债条款不利、重组失败、盈利质量恶化（概率同样存在）

在期望值为负或不确定的情况下，正确的选择是观望，而非押注。

6.1 关于盈利质量的判断标准

核心教训：当EPS与经营现金流出现显著背离时，优先相信现金流。

激进分析师在这一点上犯了一个根本性错误：他认为现金流暂时落后于会计利润是“成长型公司的正常特征”。但这个说法只有在以下条件下才成立：

  • 现金流落后是因为战略性投入（在建工程、预付款等增加）
  • 投入能够在可预期的未来转化为现金流回报

对于润泽科技，激进分析师没有提供任何证据证明这些战略性投入何时能够转化为现金流。更关键的是，扣非ROE只有11.51%说明公司的真实经营效率并不高——这不是“暂时落后”，而是“基础薄弱”。

判断原则： 纸面利润可以造假，但现金不会说谎。当两者出现显著背离时，应该假设利润存在水分，直到有充分证据证明并非如此。

6.2 关于“不确定性”的处理方式

核心教训：不确定性是双向的，不能只计算向上的期权价值。

激进分析师的核心逻辑是“催化剂未兑现=上涨空间”。这个逻辑本身没问题，但他是片面地理解了“不确定性”。

正确的处理方式：

  • 评估正面催化剂的概率和幅度
  • 评估负面催化剂的概率和幅度
  • 计算期望值，而非只考虑最好情景

对于润泽科技：

  • 正面催化剂：可转债超预期、重组成功、盈利质量验证（概率中等）
  • 负面催化剂：可转债条款不利、重组失败、盈利质量恶化（概率同样存在）

在期望值为负或不确定的情况下，正确的选择是观望，而非押注。

6.3 关于“内部人信号”的解读

核心教训：内部人的行动可能比外部人的质疑更有说服力，但前提是信息透明且动机单纯。

激进分析师把管理层回购解读为“价值背书”，但这个解读过于乐观：

  • 回购时机值得玩味：为什么不在股价更低的时候回购，而是等到从96元跌下来之后？
  • 资金配置存疑：在资产负债率62.53%、流动比率0.92的情况下，大规模回购是否明智？

判断原则： 内部人信号有价值，但需要结合信息透明度、动机纯度、资金实力来综合判断。信息不透明的内部人行动，不能作为主要决策依据。

6.4 关于“趋势”和“反转”的判断

6.1 关于盈利质量的判断标准

核心教训：当EPS与经营现金流出现显著背离时，优先相信现金流。

激进分析师在这一点上犯了一个根本性错误：他认为现金流暂时落后于会计利润是“成长型公司的正常特征”。但这个说法只有在以下条件下才成立：

  • 现金流落后是因为战略性投入（在建工程、预付款等增加）
  • 投入能够在可预期的未来转化为现金流回报

对于润泽科技，激进分析师没有提供任何证据证明这些战略性投入何时能够转化为现金流。更关键的是，扣非ROE只有11.51%说明公司的真实经营效率并不高——这不是“暂时落后”，而是“基础薄弱”。

判断原则： 纸面利润可以造假，但现金不会说谎。当两者出现显著背离时，应该假设利润存在水分，直到有充分证据证明并非如此。

6.2 关于“不确定性”的处理方式

核心教训：不确定性是双向的，不能只计算向上的期权价值。

激进分析师的核心逻辑是“催化剂未兑现=上涨空间”。这个逻辑本身没问题，但他是片面地理解了“不确定性”。

正确的处理方式：

  • 评估正面催化剂的概率和幅度
  • 评估负面催化剂的概率和幅度
  • 计算期望值，而非只考虑最好情景

对于润泽科技：

  • 正面催化剂：可转债超预期、重组成功、盈利质量验证（概率中等）
  • 负面催化剂：可转债条款不利、重组失败、盈利质量恶化（概率同样存在）

在期望值为负或不确定的情况下，正确的选择是观望，而非押注。

6.3 关于“内部人信号”的解读

核心教训：内部人的行动可能比外部人的质疑更有说服力，但前提是信息透明且动机单纯。

激进分析师把管理层回购解读为“价值背书”，但这个解读过于乐观：

  • 资金配置存疑：在资产负债率62.53%、流动比率0.92的情况下，大规模回购是否明智？

判断原则： 内部人信号有价值，但需要结合信息透明度、动机纯度、资金实力来综合判断。信息不透明的内部人行动，不能作为主要决策依据。

6.4 关于“趋势”和“反转”的判断

核心教训：趋势反转需要确认，在确认之前保持谨慎。

6.1 关于盈利质量的判断标准

核心教训：当EPS与经营现金流出现显著背离时，优先相信现金流。

激进分析师在这一点上犯了一个根本性错误：他认为现金流暂时落后于会计利润是“成长型公司的正常特征”。但这个说法只有在以下条件下才成立：

  • 现金流落后是因为战略性投入（在建工程、预付款等增加）
  • 投入能够在可预期的未来转化为现金流回报

对于润泽科技，激进分析师没有提供任何证据证明这些战略性投入何时能够转化为现金流。更关键的是，扣非ROE只有11.51%说明公司的真实经营效率并不高——这不是“暂时落后”，而是“基础薄弱”。

判断原则： 纸面利润可以造假，但现金不会说谎。当两者出现显著背离时，应该假设利润存在水分，直到有充分证据证明并非如此。

6.2 关于“不确定性”的处理方式

核心教训：不确定性是双向的，不能只计算向上的期权价值。

激进分析师的核心逻辑是“催化剂未兑现=上涨空间”。这个逻辑本身没问题，但他是片面地理解了“不确定性”。

正确的处理方式：

  • 评估正面催化剂的概率和幅度
  • 评估负面催化剂的概率和幅度
  • 计算期望值，而非只考虑最好情景

对于润泽科技：

  • 正面催化剂：可转债超预期、重组成功、盈利质量验证（概率中等）
  • 负面催化剂：可转债条款不利、重组失败、盈利质量恶化（概率同样存在）

在期望值为负或不确定的情况下，正确的选择是观望，而非押注。

6.3 关于“内部人信号”的解读

核心教训：内部人的行动可能比外部人的质疑更有说服力，但前提是信息透明且动机单纯。

激进分析师把管理层回购解读为“价值背书”，但这个解读过于乐观：

  • 回购规模不明确：一家市值1400亿的公司，回购了几千万股票，这个力度是“认可”还是“象征性表态”？
  • 回购时机值得玩味：为什么不在股价更低的时候回购，而是等到从96元跌下来之后？
  • 资金配置存疑：在资产负债率62.53%、流动比率0.92的情况下，大规模回购是否明智？

判断原则： 内部人信号有价值，但需要结合信息透明度、动机纯度、资金实力来综合判断。信息不透明的内部人行动，不能作为主要决策依据。

6.4 关于“趋势”和“反转”的判断

核心教训：趋势反转需要确认，在确认之前保持谨慎。

保守分析师关于“下降趋势中不应抄底”的警告是有意义的。虽然所有的底部都是在下降趋势中形成的，但在趋势确认反转之前，任何抄底行为都是逆势操作。

  • 等待明确的反转信号：放量突破关键阻力位、缩量止跌、底部形态形成

六、从辩论中获得的决策框架

6.1 关于盈利质量的判断标准

核心教训：当EPS与经营现金流出现显著背离时，优先相信现金流。

激进分析师在这一点上犯了一个根本性错误：他认为现金流暂时落后于会计利润是“成长型公司的正常特征”。但这个说法只有在以下条件下才成立：

  • 现金流落后是因为战略性投入（在建工程、预付款等增加）
  • 投入能够在可预期的未来转化为现金流回报

对于润泽科技，激进分析师没有提供任何证据证明这些战略性投入何时能够转化为现金流。更关键的是，扣非ROE只有11.51%说明公司的真实经营效率并不高——这不是“暂时落后”，而是“基础薄弱”。

判断原则： 纸面利润可以造假，但现金不会说谎。当两者出现显著背离时，应该假设利润存在水分，直到有充分证据证明并非如此。

6.2 关于“不确定性”的处理方式

核心教训：不确定性是双向的，不能只计算向上的期权价值。

激进分析师的核心逻辑是“催化剂未兑现=上涨空间”。这个逻辑本身没问题，但他是片面地理解了“不确定性”。

正确的处理方式：

  • 评估正面催化剂的概率和幅度
  • 评估负面催化剂的概率和幅度
  • 计算期望值，而非只考虑最好情景

对于润泽科技：

  • 正面催化剂：可转债超预期、重组成功、盈利质量验证（概率中等）
  • 负面催化剂：可转债条款不利、重组失败、盈利质量恶化（概率同样存在）

在期望值为负或不确定的情况下，正确的选择是观望，而非押注。

6.3 关于“内部人信号”的解读

核心教训：内部人的行动可能比外部人的质疑更有说服力，但前提是信息透明且动机单纯。

激进分析师把管理层回购解读为“价值背书”，但这个解读过于乐观：

  • 回购规模不明确：一家市值1400亿的公司，回购了几千万股票，这个力度是“认可”还是“象征性表态”？
  • 回购时机值得玩味：为什么不在股价更低的时候回购，而是等到从96元跌下来之后？
  • 资金配置存疑：在资产负债率62.53%、流动比率0.92的情况下，大规模回购是否明智？

判断原则： 内部人信号有价值，但需要结合信息透明度、动机纯度、资金实力来综合判断。信息不透明的内部人行动，不能作为主要决策依据。

6.4 关于“趋势”和“反转”的判断

核心教训：趋势反转需要确认，在确认之前保持谨慎。

保守分析师关于“下降趋势中不应抄底”的警告是有意义的。虽然所有的底部都是在下降趋势中形成的，但在趋势确认反转之前，任何抄底行为都是逆势操作。

正确的处理方式：

  • 不要试图“接飞刀”——在趋势明朗之前，减少暴露
  • 等待明确的反转信号：放量突破关键阻力位、缩量止跌、底部形态形成
  • 如果要参与，使用严格的止损保护

对于润泽科技： 当前技术面显示下降趋势未变，80元整数关口岌岌可危。在这种情况下，即使认为“跌到位了”，也应该等待确认信号，而非提前入场。

指标说明

股价曲线 = 当天用于回放估值/执行的价格口径；当前日级回放默认使用 14:55 附近的可见价格或当日可用标记价格。
持仓金额曲线 = 日末持股数 × 当天股价。
真实总账户净值 = 真实账户现金 + 全部真实持仓市值。这是执行层真正发生买卖和记账的总账户口径。
真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。
策略拆分盈亏 = 策略覆盖资产 - 累计分配资金。
策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
