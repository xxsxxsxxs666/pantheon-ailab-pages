七、从过去的错误中学习

我在过去类似情况下犯过哪些错误？

错误一：把信息不透明当作机会

“信息不透明有时候恰恰意味着机会大于风险”——这句话在成熟的机构市场或许成立，但对散户来说，信息不对称永远是劣势，不是优势。我们以为“市场不知道的真实情况”往往比我们知道的情况更糟，而不是更好。

错误二：逆势抄底的沉没成本谬误

在下降趋势中不断补仓，试图摊薄成本。结果是：趋势没有改变，但本金已经大幅亏损。“补仓是不是陷阱，取决于你补的是什么股票”——这话听起来有道理，但如果我们不能确定股票的质量，补仓只会放大损失。

错误三：过于相信“支撑位”

17元被多次测试但从未能企稳。每一次“支撑位”都是用来被跌破的，而不是用来抄底的。

错误四：把猜测当事实

把“经营范围变更 + 担保公告 + 临时股东大会”强行拼凑成“重大资产重组前奏”，没有一条实质性证据。这种故事在A股市场经常被庄家利用来出货。

这次我将如何避免这些错误？

  1. 不把信息不透明当作机会——等待信息明朗
  2. 尊重趋势——在趋势改变之前，不逆势操作
  3. 仓位控制——即使判断错误，也不会导致灾难性损失
  4. 具体化触发条件——不是猜测，而是设定明确的观察指标
  5. 分情景应对——不是非此即彼，而是根据不同情况准备不同方案

持仓金额曲线 = 日末持股数 × 当天股价。
真实总账户净值 = 真实账户现金 + 全部真实持仓市值。这是执行层真正发生买卖和记账的总账户口径。
真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。
策略拆分盈亏 = 策略覆盖资产 - 累计分配资金。
策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
