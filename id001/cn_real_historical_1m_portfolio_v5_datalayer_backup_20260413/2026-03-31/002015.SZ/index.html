五、从辩论中学习的关键教训

lesson 1: “可能”不是买入的理由

激进分析师的分析充斥着“可能转型”、“可能有新业务”、“可能受益政策”。这些假设没有任何数据支撑，它们是愿望清单，不是投资依据。

经验： 在信息不对称时盲目逆向投资，是最危险的赌注行为。真正的逆向投资大师（巴菲特）建立在对基本面的深度理解上，而非基于“可能有利好”。

lesson 2: 流动性风险是被低估的风险

止损位16.50元在理论上是1.26%的损失，但实际执行时可能因流动性枯竭而无法成交。激进分析师的“有限风险”假设在实践中不成立。

经验： 在评估风险时，必须考虑最坏情景下的实际执行难度，而非理论上的最优成交价。

lesson 3: 担保频率是资金压力的先行指标

一个月三次担保公告，这个信号的重要性被低估。如果公司财务状况良好，为什么需要如此频繁地为子公司提供担保？

经验： 公告频率本身就是信息。当正面消息和负面信号同时出现时，应该更重视负面信号。

lesson 4: 中性观点有时是逃避责任的托词

中性分析师的“我们需要更多数据”听起来很理性，但它没有给出明确的方向性建议。在投资决策中，模糊的正确不如明确的错误——至少后者可以验证和修正。

经验： 等待也需要理由。如果等待的原因只是“怕做错”，这本身就是一个需要克服的心理障碍。

股价曲线 = 当天用于回放估值/执行的价格口径；当前日级回放默认使用 14:55 附近的可见价格或当日可用标记价格。
持仓金额曲线 = 日末持股数 × 当天股价。
真实总账户净值 = 真实账户现金 + 全部真实持仓市值。这是执行层真正发生买卖和记账的总账户口径。
真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。
策略拆分盈亏 = 策略覆盖资产 - 累计分配资金。
策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
