更重要的是，42.45元的前期低点距离现价只有3.7%的空间。这意味着下行风险是有限的，而上行空间（哪怕按照保守的43元目标价）也有一定的赔率。

完全空仓不是风险管理，是逃避责任。

第三，中性分析师的分阶段策略是最优解

中性分析师的核心贡献是：承认不确定性，但不完全回避风险。

分三步建仓的策略完美平衡了双方关切：

  • 第一步10%-15%建仓，既保留参与机会，又控制下行风险
  • 第二步等待铜价信号，是理性的右侧确认
  • 第三步等待技术面突破，是纪律性的风险管理

最终25%-35%的仓位，既不是激进派的50%，也不是安全派的10%-20%，而是根据实际风险收益比动态调整的结果。

建仓策略

立即执行：在当前价格44.10元附近建仓10%-12%。

加仓条件（任选其一即可加仓5%-10%）：

  • 铜价企稳8000美元并出现反弹迹象
  • 股价回调至43.5-44元区间
  • 成交量明显放大

禁止加仓：铜价跌破7500美元时不加仓；股价跌破43元时不加仓。

止损设置

硬止损：42元整数关口。一旦有效跌破，必须清仓离场。这个位置下方支撑薄弱，可能直接测试40元。

软止损：45.5-46元区间出现滞涨时，减仓一半锁定利润。

止盈策略

  • 股价触及47-48元：减仓三分之一
  • 股价触及45.5-46元：减仓一半
  • 股价触及48-50元区间：考虑清仓

仓位上限

单一标的仓位不超过30%。在任何情况下，不因为对这个标的的看好而突破这个限制。

  1. 铜价持续下跌风险：如果铜价跌破7500美元，净利率可能被压缩至盈亏平衡点，股价将面临更大压力。

  4. 系统性风险：如果A股市场整体下跌，江西铜业难以独善其身。

结语

三位分析师代表了投资决策的三个维度：进攻、防守、平衡。

激进派看到了机会，但低估了风险；安全派控制了风险，但忽视了机会成本；中性派在两者之间找到了最优解。

投资不是非黑即白的判断题，而是在不确定性中寻找最优概率的解答题。

我的建议是：持有，但带着明确的行动方案。10%-15%的仓位是“试错成本”，而不是“赌博筹码”。等待信号确认后再加仓，这是在不确定性市场中生存和发展的正确方式。

股价曲线 = 当天用于回放估值/执行的价格口径；当前日级回放默认使用 14:55 附近的可见价格或当日可用标记价格。
持仓金额曲线 = 日末持股数 × 当天股价。
真实总账户净值 = 真实账户现金 + 全部真实持仓市值。这是执行层真正发生买卖和记账的总账户口径。
真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。
策略拆分盈亏 = 策略覆盖资产 - 累计分配资金。
策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
