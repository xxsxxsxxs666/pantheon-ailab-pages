    • 重组方案未知，无法评估价值
    • 财务数据未知，无法估值底部

  2. 不确定性 ≠ 单边机会
    激进分析师只计算了"重组成功+方案超预期"的上涨情景，忽略了"重组失败+方案不及预期"的下跌情景。在综合置信度仅0.37的情况下，完全押注乐观情景是不负责任的。

为什么不是"卖出"？

保守分析师的核心错误在于：把"不确定性"解读为"负面信号"。

这个解读同样存在问题：

  1. 沉默 ≠ 负面
    重组方案需要经过内部审批、尽职调查、交易对方协商后才能披露，筹备期长不等于遇到障碍

  2. 踏空也是成本
    如果重组成功且超预期，股价可能从80元涨至95元甚至更高。完全空仓意味着放弃一个可能的高胜率机会

为什么是"持有（1成观察仓）"？

核心理由：

  1. 存在实质性正面催化剂
    回购+重组双轮驱动，说明管理层有战略诉求。这个判断不依赖具体数据，而是基于行为逻辑。

  2. 下行风险有限
    80元整数关口虽然"脆弱"，但至少在当前形成心理支撑。即使重组失败，向下空间（至75元约7%）也相对可控。

  3. 保留获利可能性
    1成仓位可以在重组成功时分享收益，避免踏空；在重组失败时损失有限（最大损失约7%）。

  4. 动态调整空间
    中性分析师提出的"信号验证"框架值得借鉴。1成仓可以在获取更多信息后灵活加减。

四、交易计划细化

阶段一：建仓期（第1周）

买入价：80-81元
仓位：1成（10%）
占组合比例：低风险仓位

阶段二：观察验证期（第2-6周）

触发加仓的条件（仓位从1成增至2-3成）：

  • 重组预案正式披露，标的资产盈利前景明确
  • 回购完成公告显示实际回购量≥流通股本1%
  • 股价放量突破82.50元并连续3日站稳
  • 财务数据披露后显示基本面稳健

触发止损的条件（清仓离场）：

  • 股价有效跌破77元（跌幅约4%）
  • 重组方案披露后大幅低于预期
  • 出现任何重大负面信息（诉讼、监管函、业绩预警）

触发减仓的条件（仓位从1成降至0.5成）：

  • 股价反弹至85元附近但量能萎缩
  • 重组审批周期延长
  • 大盘系统性下跌

根据获取的完整信息，做最终决策：

  • 若重组成功且方案超预期：持有至90-95元
  • 若重组成功但方案一般：持有至85元附近了结

六、辩论经验总结与决策改进

从过去类似案例中学到的教训

  1. "朦胧期"未必是最佳买点
    历史经验表明，许多重组股在预案公布后反而出现"利好兑现"走势，尤其是监管趋严后，概念炒作的持续性大幅下降。

  2. 财务数据缺失期间往往隐藏风险
    许多"突然"曝出财务问题的公司，在数据真空期看似平静。完全忽视基本面信息是危险的。

  3. 仓位管理比方向判断更重要
    在不确定性环境下，控制仓位本身就是风险管理。1成仓的错误成本（最大7%）远低于3-5成仓（最大15-35%）。

本次决策的改进

相比之前类似场景的改进：

  • 避免了两个极端（全进/全出）
  • 明确给出了可执行的量化标准
  • 保留了动态调整空间
  • 计算了期望收益而非单情景收益

相比单纯采用某一方观点的改进：

  • 接受激进分析师的"催化剂存在"观点（但降低权重）
  • 接受保守分析师的"尾部风险"警示（但给予合理权重）
  • 采纳中性分析师的"仓位管理"框架（这是最宝贵的洞见）

真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。
策略拆分盈亏 = 策略覆盖资产 - 累计分配资金。
策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
