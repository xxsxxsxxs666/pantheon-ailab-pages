六、风险清单与应对

风险一：财务数据缺失风险（高）

描述： 在财务数据缺失6个月以上的情况下投资，无法评估公司真实价值。

应对： 将仓位控制在10%以下，等待财务数据披露后再做重大决策。

风险二：融资杠杆踩踏风险（中）

描述： 融资余额持续增长意味着一旦股价回调，融资盘集中平仓可能放大跌幅。

应对： 密切关注融资余额变化，日降幅超过10%时减仓至5%以下。

风险三：信息不对称风险（中）

描述： 公司主动发布异常波动公告和风险提示，可能存在重大信息未披露。

应对： 对任何公司公告保持高度警惕，负面公告后立即评估是否需要减仓。

风险四：流动性风险（低）

描述： 如果公司基本面恶化，可能出现流动性枯竭，无法以合理价格卖出。

应对： 任何时候都保留至少30%的现金，避免全仓被套。

七、基于历史经验的改进

过去常见的错误

错误一：过早抄底

过去的教训是：在信息不透明时过度依赖技术信号和资金流向，忽视了基本面数据的缺失本身就是风险信号。

本次改进： 不因为“感觉跌够了”就盲目抄底，仓位控制在10%以下，严格执行止损。

错误二：混淆市场数据与个股数据

过去常常把市场整体的涨跌当作个股涨跌的原因或证据。

本次改进： 只使用协鑫能科个股的数据，拒绝使用全市场数据为个股背书。

错误三：追求完美时机

过去常常想等到“最佳买点”才入场，结果错过整个行情。

本次改进： 接受不确定性，在控制仓位的前提下有限参与，而不是完全空仓错过机会。

指标说明

股价曲线 = 当天用于回放估值/执行的价格口径；当前日级回放默认使用 14:55 附近的可见价格或当日可用标记价格。
持仓金额曲线 = 日末持股数 × 当天股价。
真实总账户净值 = 真实账户现金 + 全部真实持仓市值。这是执行层真正发生买卖和记账的总账户口径。
真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。
策略拆分盈亏 = 策略覆盖资产 - 累计分配资金。
策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
