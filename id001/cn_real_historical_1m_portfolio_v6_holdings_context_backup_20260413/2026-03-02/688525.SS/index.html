三、决策依据详解

1. 为什么卖出而非持有？

持有只在一种情况下合理：投资者确信股价将在一段时间内维持高位，且有能力承受50%以上的潜在下跌，同时有充足流动性应对极端情况。

但这种假设对于99%的投资者都不成立。

支持卖出的关键数据链：

当前PE 465倍 → 隐含2026年利润5亿元（乐观预测）
         → 但年报尚未披露，预期可能进一步下调
         → 即便实现5亿利润，合理PE 40倍 → 目标价40元
         → 当前160元意味着还有75%下跌空间
         → 不确定性和风险都指向同一方向：下行

为什么不是"持有观望"？

年报披露在即（3月中上旬），这是典型的信息不确定窗口。历史经验表明，绩差股在高估值、高情绪状态下，一旦遭遇财报冲击，下跌速度极快，且伴随流动性枯竭。"持有等待"本质上是把决定权交给不确定的外部事件，这不是风险管理，是赌博。

2. 为什么卖出而非买入？

空仓投资者的潜在动机是"抄底"。但这个想法有三个致命问题：

  • 没有"铁底"保证：120元不是底部，100元、80元、50元都可能在悲观情景下触及
  • 机会成本巨大：在此期间，正確的做法是配置其他低估资产获取正收益
  • 时间价值损耗：资金被套在佰维存储，就失去了在其他地方赚钱的机会

3. 为什么卖出而非等待更低价格？

理论上"等跌到100元再买"听起来合理，但存在两个问题：

  • 时间窗口不确定：股价可能在年报前维持高位1-2个月
  • 情绪反转不可预测：如果年报不是灾难性的，股价可能根本不跌

结论：对于空仓投资者，最优策略是不参与，而非"等待抄底"。

指标说明

股价曲线 = 当天用于回放估值/执行的价格口径；当前日级回放默认使用 14:55 附近的可见价格或当日可用标记价格。
持仓金额曲线 = 日末持股数 × 当天股价。
真实总账户净值 = 真实账户现金 + 全部真实持仓市值。这是执行层真正发生买卖和记账的总账户口径。
真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。
策略拆分盈亏 = 策略覆盖资产 - 累计分配资金。
策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
