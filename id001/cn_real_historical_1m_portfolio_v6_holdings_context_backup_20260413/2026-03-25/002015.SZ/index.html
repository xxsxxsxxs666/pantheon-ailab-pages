二、激进分析师论证的关键缺陷

缺陷一：选择性使用证据

激进分析师只考虑了"数据缺失可能隐藏利好"这一种可能，完全忽视了数据缺失同样可能隐藏灾难性利空。

保守分析师的反驳精准到位：

"看看A股历史，有多少'大集团'旗下的上市公司最后让投资者血本无归？恒大旗下的股票、海航系、泛海系，哪一个不是声名显赫？"

协鑫能科是独立法人，财务报表独立于母公司。在没有看到年报之前，任何关于"协鑫集团实力雄厚"的论据都不能直接套用在这只股票上。

缺陷二：对技术面的误读

激进分析师说"技术指标是滞后指标"，但他自己使用RSI超卖作为买入信号，恰恰也是在用滞后指标做预测。更严重的是，他对量价关系的解读存在明显矛盾：

他一边说"缩量下跌说明抛压不重，主力控盘明显"，一边又说"上涨时放量"。但实际情况是：3月12-13日放量上涨后，3月16日以来持续缩量下跌。这说明什么？说明3月12-13日的放量更可能是短线资金借机出货，而非新主力入场建仓。

缺陷三：高估风险收益比

激进分析师声称"向上的空间是向下的3到5倍"，但这个计算的前提是"向上概率大于向下概率"。然而，从当前技术面看：

  • MACD在零轴下方运行
  • 股价处于下降通道中

在这种情况下，向上和向下的概率哪个更大？

中性分析师给出了更客观的评估：

"潜在收益约5%到8%，这样赔率大约是1比1到1比1.5，虽然不如激进分析师说的1比3那么高，但考虑到不确定性较大，这个赔率反而更现实。"

真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。
策略拆分盈亏 = 策略覆盖资产 - 累计分配资金。
策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
