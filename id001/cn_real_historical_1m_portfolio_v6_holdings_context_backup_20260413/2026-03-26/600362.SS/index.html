关于"持有"决策的反思

我知道"持有"可能被批评为没有立场。但在这个案例中，持有是基于对不确定性的理性认知，而不是懦弱。

从过去的经验教训来看：

历史上，许多投资者在"利好兑现"时追高买入，最终被套在山顶。2015年杠杆牛泡沫破裂、2021年核心资产崩塌，太多案例证明：在信息不对称时盲目乐观，往往付出惨重代价。

同时，那些在所有利好出尽、股价反而下跌后仍然坚守的人，往往成为最后的"接盘侠"。

因此，最优策略是：

  • 不在朦胧利好时追涨
  • 不在利好兑现、股价反跌时盲目抄底
  • 等待市场给出明确的方向信号后再行动
  • 在等待期间保持弹药充足

关于激进分析师的"赔率论"：

"赔率1:3到1:4的机会，你们不敢要？"

我的回答是：高赔率的前提是概率不被严重高估。 当一只股票在利好兑现后反而暴跌，市场的定价行为已经告诉我们：有人知道我们不知道的东西。在这种情况下，所谓的"高赔率"可能只是一个陷阱。

关于保守分析师的"七七定律"：

"海外并购有个说法，叫'七七定律'——70%的并购失败，70%失败案例中又有70%死于整合过程。"

这个统计值得重视。历史上，中国企业海外矿业并购失败的案例远多于成功案例。在没有更多信息的情况下，保守一点没有坏处。

关于中性分析师的"试探性建仓"：

这个思路有合理性，但在当前情境下，即使要试探，仓位也应该更低（3%-5%），止损更严格（43元），时间限制更短（一周内无信号就离场）。

结论

明确建议：持有，等待更明确的信号。

在当前信息环境下：

  • 44元可能是支撑，但支撑不一定牢靠
  • 收购可能是利好，但利好兑现不等于股价上涨
  • 长期逻辑可能正确，但短期操作需要耐心

最佳策略是保持耐心，等待市场给出明确答案。 真正的Alpha不在于抓住每一个机会，而在于识别哪些机会值得冒风险。

股价曲线 = 当天用于回放估值/执行的价格口径；当前日级回放默认使用 14:55 附近的可见价格或当日可用标记价格。
持仓金额曲线 = 日末持股数 × 当天股价。
真实总账户净值 = 真实账户现金 + 全部真实持仓市值。这是执行层真正发生买卖和记账的总账户口径。
真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。
策略拆分盈亏 = 策略覆盖资产 - 累计分配资金。
策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
