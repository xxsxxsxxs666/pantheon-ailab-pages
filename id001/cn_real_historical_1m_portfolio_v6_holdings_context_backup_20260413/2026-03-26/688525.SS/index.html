激进分析师的致命缺陷

第一，概率模型是主观构建，非客观分析。

"乐观情景（概率25%+）：280-320元"

这个25%的概率从何处得来？激进分析师没有提供任何统计基础。这本质上是先射箭后画靶——先设定"应该买入"的结论，再为这个结论赋予看似合理的概率权重。

第二，"为什么不涨到1000元"的反驳同样适用于自己。

如果1-2月数据真的能年化到90亿利润，市场没有理由在240元徘徊。这个质疑指向的是激进分析师的估值方法，而非市场的错误定价。

第三，忽视了合同的双向风险。

锁定价格=锁定成本+锁定风险。当晶圆价格下跌时，固定价格合同反而成为竞争劣势来源。激进分析师只强调上涨情景，忽略了下跌情景的不对称性。

保守分析师的致命缺陷

第一，"把可能跌破当成已经跌破"的逻辑谬误。

238.88元支撑目前尚未有效跌破。在支撑被突破之前，它就是有效的。这是技术分析的基本原则，不能用"可能"来代替"事实"。

第二，银行授信解读过于负面。

银行风控部门的风控能力通常强于散户投资者。200亿授信至少说明银行认为公司具备还款能力，而非仅凭"银行想赚利息"就能解释。

第三，目标价180-200元同样缺乏依据。

只给出下跌目标价而不分析上涨空间，是另一种形式的选择性忽视。如果公司真的能实现年化60亿利润，200元股价意味着不到4倍PE，在行业复苏期这个估值偏低。

四、基于历史经验教训的决策优化

关键教训一：周期股估值陷阱是真实存在的

2015年创业板泡沫、2019年猪肉股周期、2021年新能源周期——每一次市场都犯同样错误：用成长股逻辑给周期股定价。佰维存储若被当作成长股炒作，下一轮景气下行时将面临估值与业绩的"双杀"。

决策影响：不能基于"可能达到90亿年化利润"给予成长股估值。即使买入，也要以周期股逻辑定价。

关键教训二：技术面不是决策的唯一依据，但提供了关键的风险管理框架

2022年光伏板块崩盘、2023年AI概念回调——每一次市场见顶前都伴随着清晰的技术面背离。当前佰维存储从258元回落至240元，跌幅7%，需等待技术面企稳信号。

决策影响：设置严格止损位（225元），跌破必须执行，不幻想"V型反转"。

关键教训三：共识过于乐观时往往见顶，过于悲观时往往见底

社交媒体情绪评分8.8/10，媒体正面率9.5/10——这是警示信号而非买入信号。当所有人都看好时，后续买入力量已经衰竭。

决策影响：不追高，不在情绪高点加仓。

关键教训四：仓位管理比方向判断更重要

无论看多还是看空，20%以上单一股票仓位在高波动市场中风险极高。2023年数据显示，A股单只股票持仓超过20%的投资者，回撤幅度普遍超过组合投资者。

决策影响：15%仓位上限，跌破支撑不补仓。

股价曲线 = 当天用于回放估值/执行的价格口径；当前日级回放默认使用 14:55 附近的可见价格或当日可用标记价格。
持仓金额曲线 = 日末持股数 × 当天股价。
真实总账户净值 = 真实账户现金 + 全部真实持仓市值。这是执行层真正发生买卖和记账的总账户口径。
真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。
策略拆分盈亏 = 策略覆盖资产 - 累计分配资金。
策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
