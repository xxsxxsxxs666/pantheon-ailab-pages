根本性错误：

  1. "数据缺失≠负面数据"是伪命题

    激进分析师原话："数据缺失期往往孕育着最大的机会"

    这是危险的逻辑跳跃。机会和陷阱在数据缺失时长得完全一样——你凭什么判断这是机会而非陷阱？用希望填补信息空白是散户思维，不是专业分析。

  2. 把集团光环套用到子公司是投资谬误

    激进分析师原话："协鑫集团的背景是保利协鑫——全球硅料龙头！"

    保利协鑫是港股上市公司，协鑫能科是A股上市公司。法律上它们是独立法人，你不能把母公司的信用背书直接套在子公司身上。要是能直接用，公告里直接写"集团担保"就行了，何必单独发担保公告？

  3. 6:1赔率忽视胜率是数学错误

    激进分析师原话："下跌5%对上涨30%，赔率是6:1！"

    赔率必须结合胜率才有意义。如果这个赔率的胜率只有20%，期望收益是负的（0.2×30% - 0.8×5% = 2%），这仍然是个糟糕的赌注。

  4. 技术指标不可靠却声称"超跌反转信号"

    激进分析师原话："MACD、KDJ进入超卖区间，这是反转信号"

    技术分析报告白纸黑字写着：缺少成交量数据，MACD、KDJ、RSI、布林带全部无法可靠计算。你自己承认指标不可靠，又用不可靠的指标来支撑"反转信号"，这是自相矛盾。

保守分析师：方向正确但操作过于保守

最强论点：

  1. 财务数据全面缺失是重大风险信号

    保守分析师原话："不知道的，就不要假设它不存在。"

    这话虽然保守，但在资本市场里，保守就是保命。PE、PB、ROE、营收、利润、现金流——六个核心指标全部"不可得"，这在正常市场环境下不可接受。

  2. 异常波动公告连发两条是公司内部的风险提示

    保守分析师原话："公司自己认为需要向投资者提示风险。"

    根据A股规则，连续3个交易日涨跌幅偏离值超过20%才需要发布异常波动公告。公司短期内连发两条，说明波动幅度和频率都非常大。公司内部的判断，比外部分析师的推测更可信。

  3. 担保公告密集是风险信号而非扩张信号

    保守分析师原话："担保公告密集发布本身就值得警惕。"

    银行愿意贷款和股票值得买是两码事。担保公告密集发布说明公司需要额外信用增级措施，这不是实力强的象征，而是融资成本高的信号。

    保守分析师原话："没有量能配合的反弹能走多远？"

    技术分析报告明确指出缺少成交量数据。激进分析师口中的"放量反弹"没有任何数据支撑。

二、三位分析师核心论点评估

激进分析师：论点有力但根本前提错误

最强论点：

  • 涨停板确实需要真金白银，52只涨停股之一说明短线资金有动作
  • 广西金控合作框架是实质性利好

根本性错误：

  1. "数据缺失≠负面数据"是伪命题

    激进分析师原话："数据缺失期往往孕育着最大的机会"

    这是危险的逻辑跳跃。机会和陷阱在数据缺失时长得完全一样——你凭什么判断这是机会而非陷阱？用希望填补信息空白是散户思维，不是专业分析。

  2. 把集团光环套用到子公司是投资谬误

    激进分析师原话："协鑫集团的背景是保利协鑫——全球硅料龙头！"

    保利协鑫是港股上市公司，协鑫能科是A股上市公司。法律上它们是独立法人，你不能把母公司的信用背书直接套在子公司身上。要是能直接用，公告里直接写"集团担保"就行了，何必单独发担保公告？

  3. 6:1赔率忽视胜率是数学错误

    激进分析师原话："下跌5%对上涨30%，赔率是6:1！"

    赔率必须结合胜率才有意义。如果这个赔率的胜率只有20%，期望收益是负的（0.2×30% - 0.8×5% = 2%），这仍然是个糟糕的赌注。

  4. 技术指标不可靠却声称"超跌反转信号"

    激进分析师原话："MACD、KDJ进入超卖区间，这是反转信号"

    技术分析报告白纸黑字写着：缺少成交量数据，MACD、KDJ、RSI、布林带全部无法可靠计算。你自己承认指标不可靠，又用不可靠的指标来支撑"反转信号"，这是自相矛盾。

保守分析师：方向正确但操作过于保守

最强论点：

  1. 财务数据全面缺失是重大风险信号

    保守分析师原话："不知道的，就不要假设它不存在。"

    这话虽然保守，但在资本市场里，保守就是保命。PE、PB、ROE、营收、利润、现金流——六个核心指标全部"不可得"，这在正常市场环境下不可接受。

  2. 异常波动公告连发两条是公司内部的风险提示

    保守分析师原话："公司自己认为需要向投资者提示风险。"

    根据A股规则，连续3个交易日涨跌幅偏离值超过20%才需要发布异常波动公告。公司短期内连发两条，说明波动幅度和频率都非常大。公司内部的判断，比外部分析师的推测更可信。

  3. 担保公告密集是风险信号而非扩张信号

    保守分析师原话："担保公告密集发布本身就值得警惕。"

    银行愿意贷款和股票值得买是两码事。担保公告密集发布说明公司需要额外信用增级措施，这不是实力强的象征，而是融资成本高的信号。

  4. 无量反弹是下跌中继

    保守分析师原话："没有量能配合的反弹能走多远？"

    技术分析报告明确指出缺少成交量数据。激进分析师口中的"放量反弹"没有任何数据支撑。

股价曲线 = 当天用于回放估值/执行的价格口径；当前日级回放默认使用 14:55 附近的可见价格或当日可用标记价格。
持仓金额曲线 = 日末持股数 × 当天股价。
真实总账户净值 = 真实账户现金 + 全部真实持仓市值。这是执行层真正发生买卖和记账的总账户口径。
真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。
策略拆分盈亏 = 策略覆盖资产 - 累计分配资金。
策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
