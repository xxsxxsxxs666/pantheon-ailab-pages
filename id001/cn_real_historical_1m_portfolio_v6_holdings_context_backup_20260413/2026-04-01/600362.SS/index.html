  1. 期望收益计算存在根本缺陷——只呈现最好情景下的收益，没有加权概率分布。中性分析师的期望收益测算（50%概率涨2.7%，30%概率跌2.9%）更接近现实，期望收益仅0.48%，扣除交易成本后接近零
  2. "数据缺失是机会"的逻辑悖论——在信息不对称市场中，机构获取信息的能力远强于散户，数据缺失更可能意味着我们处于劣势而非优势
  3. "机构低位吸筹"纯属主观臆断——无任何持仓数据、大宗交易记录或机构调研公告支撑此假设

1.2 保守分析师的论证得失

最强论点：

  • 净利润增速(2.41%)与营收增速(5.42%)的巨大差距，揭示了毛利率恶化趋势，这对周期股是实质性风险警示
  • PE、PB、ROE、资产负债率等核心指标全部缺失，估值基础确实薄弱——激进分析师声称的"PE约18.9倍"恰恰是报告中标注为"不可得"的数据
  • 期望收益的量化分析揭示：在没有量能确认的情况下，上涨概率被高估

过度悲观之处：

  1. "内部控制无重大缺陷"不只是及格线，对于年营收5400亿的大规模企业，内部控制有效性是公司治理健康的正面信号
  2. 43元支撑位已被多次测试，其有效性应被赋予更高权重，而非简单假设30%概率跌回
  3. 过度强调"等待完美信号"可能导致完全踏空趋势性行情

1.3 中性分析师的论证得失

最大贡献：

  • 指出两方共同忽视的关键变量：铜价走势和宏观大盘环境
  • 区分短期择时信号与中期战略配置的逻辑框架
  • 提出"动态调整而非静态观望"的务实策略

不足之处：

  • 未深入分析铜价周期位置，而这是周期股投资最核心的变量
  • 对分红提升的解读略显中性，应给予更多正面权重

三、主席裁定：持有（轻仓观望）

3.1 裁定的核心理由

第一，数据缺失是客观约束，不可绕过。

激进分析师试图将"数据不可得"转化为"机会所在"，但这是危险的逻辑跳跃。PE、PB、资产负债率、现金流等核心指标缺失，意味着我们无法判断：

  • 公司估值在行业中是偏高还是偏低
  • 净利润增长的质量（是真金白银还是会计调整）
  • 债务压力是否可控
  • 铜价下行时的抗风险能力

没有这些信息，任何"低估"或"机会"的判断都是主观臆断。

第二，技术信号的模糊性要求谨慎。

当日3%的涨幅没有成交量数据支撑。技术报告明确指出无法计算均线、RSI、MACD等指标。历史反复证明：无量反弹是A股市场最危险的陷阱之一，随后往往是加速下行。

第三，基本面出现令人不安的警讯。

营收增长5.42%，净利润仅增长2.41%——这个差距意味着毛利率在恶化。对于铜价高度敏感的周期股，这可能预示着：

  • 铜价下行压力开始侵蚀利润
  • 成本刚性上升
  • 下游需求可能走弱

在核心数据缺失的情况下，这个警讯的分量更重而非更轻。

第四，这不是"两边都不得罪"的中庸选择。

裁定的依据是：

  • 重仓出击没有正期望收益（期望值-0.15%）
  • 但完全空仓也不合理（43元支撑有效，低位盲目看空同样危险）
  • 5-10%的试探性仓位是在不确定性中保持市场接触、控制下行风险的理性选择

3.2 反对"持有作为后备选择"的质疑

如果持有只是因为"两边都有道理"，这就是错误的决策模式。但本裁定的逻辑是：

  • 激进策略的期望收益为负，不支持重仓
  • 保守策略完全踏空风险也不可取，因为43元支撑有效
  • 5-10%仓位是唯一具有正期望的选项——亏损有限（≤4%），但保留了参与行情的可能性

5.1 激进分析师的教训

错误模式："锚定历史高点"

  • 激进分析师锚定48.26元（52周高点），认为44元是低位
  • 教训：过去价格不是价值依据。2007年中石油从48元跌到10元以下的案例历历在目
  • 本决策：不以历史高点为参照，以当前信息质量评估价值

错误模式："把愿景当证据"

  • 激进分析师将新能源长期需求作为短期买入依据
  • 教训：从长期逻辑到短期交易之间有巨大的不确定性鸿沟
  • 本决策：长期逻辑存在，但只在仓位控制中给予有限权重（最多5%），不作为重仓依据

5.2 保守分析师的教训

错误模式："数据完美主义"

  • 保守分析师认为数据不完整就完全不可评估
  • 教训：完全等待完美信息可能导致永远无法行动
  • 本决策：承认数据不足，但现有数据（营收5446亿、净利润71亿、分红提升）足以支持极轻仓试探

5.3 主席的自我反思框架

  1. 避免"幸存者偏差"——不要只看到成功抄底的案例，忽视大量抄底失败的教训
  2. 警惕"确认偏误"——不要只寻找支持买入的信息，而忽视看跌信号的合理性
  3. 控制"过度自信"——在数据极度有限的情况下，谨慎决策优于大胆押注
  4. 区分"故事"与"证据"——新能源长期利好是真实存在的故事，但不能替代短期技术面和基本面的证据

股价曲线 = 当天用于回放估值/执行的价格口径；当前日级回放默认使用 14:55 附近的可见价格或当日可用标记价格。
持仓金额曲线 = 日末持股数 × 当天股价。
真实总账户净值 = 真实账户现金 + 全部真实持仓市值。这是执行层真正发生买卖和记账的总账户口径。
真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。
策略拆分盈亏 = 策略覆盖资产 - 累计分配资金。
策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
