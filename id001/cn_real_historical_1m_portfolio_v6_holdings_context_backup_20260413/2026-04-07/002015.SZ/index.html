  1. 期望值计算中"做对概率40%"缺乏充分依据——更应该是45-50%的中性概率
  2. 对催化剂的处理过于武断——完全排除广西金控合作的可能性

  • 未明确回答：当前16.36元价格究竟是该观望还是该轻仓？

五、决策背后的核心理念

为什么不是"持有"？

持有意味着继续承受风险但不获得收益。在当前技术面偏空、基本面有警示信号的情况下，"持有"只是把问题推迟，无法解决任何问题。

为什么不是"买入"？

期望值为负。数学不会说谎——在一个期望值为负的交易上建仓，不是在投资，是在赌博。

为什么是"卖出"？

  1. 保护本金：在不确定性高的环境下，本金比机会更重要
  2. 等待更好的价格：13.50-14.00元的超跌区间将提供更好的风险收益比
  3. 等待更明确的信号：广西金控合作的具体项目落地、或者公司主动澄清异常波动公告的原因

从历史经验中学习

  • 2015年股灾教训：主力资金大举买入不代表安全，下跌趋势中的反弹往往是离场机会
  • "战略合作"炒作教训：大量"战略合作协议"最终没有落地，跟风买入者往往成为最后买单者
  • 高换手率日后的走势规律：涨停日高换手率配合后期无法突破，是主力派发的经典形态

六、给交易员的一句话总结

协鑫能科当前不是"买不买"的问题，而是"要不要跑"的问题。风险收益比不支持做多，技术面不支持做多，基本面警示信号明确。建议卖出，等待更好的机会。真正的交易高手不是"什么时候都敢买"，而是"知道什么时候不该买"。

股价曲线 = 当天用于回放估值/执行的价格口径；当前日级回放默认使用 14:55 附近的可见价格或当日可用标记价格。
持仓金额曲线 = 日末持股数 × 当天股价。
真实总账户净值 = 真实账户现金 + 全部真实持仓市值。这是执行层真正发生买卖和记账的总账户口径。
真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。
策略拆分盈亏 = 策略覆盖资产 - 累计分配资金。
策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
