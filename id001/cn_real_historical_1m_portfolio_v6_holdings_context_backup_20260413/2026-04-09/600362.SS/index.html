二、关键分歧点裁决

争议1：净利率1.31%是硬伤还是行业特性？

裁决：部分支持保守分析师，但认可中性分析师的动态观点

激进分析师说"矿业公司净利率低是常态"，这话有道理。但问题在于：如果净利率长期维持1.31%，意味着规模扩张并不创造股东价值。这不是行业特性，而是盈利质量警示。

关键观察点：需要2026年中报验证净利率是否改善。若铜价维持高位而净利率继续恶化，则确认"增收不增利"是结构性问题而非周期现象。

争议2：技术面是"主力建仓"还是"游资短炒"？

裁决：支持保守分析师的判断

激进分析师的"主力洗盘吸筹"论缺乏证据支持。真正的机构建仓通常有以下特征：

  • 持续数周至数月的稳定买入
  • 股价波动较小，呈现"慢牛"走势
  • 成交量温和放大而非脉冲式

4月8日-9日的走势（6.39%冲高→次日1.38%回落，成交量萎缩）更符合游资特征：快速拉升吸引跟风盘，次日兑现利润。

三、历史经验教训应用

教训1：避免"叙事陷阱"

激进分析师的"新能源超级周期"叙事确实激动人心，但好故事≠好股票。2021年新能源行情顶峰时，市场同样充满"十年最确定性机会"的声音，结果呢？

本次决策规避：不因长期逻辑看好而忽视短期风险信号。

教训2：警惕"利好过度报道"

社保持仓被报道5次以上，可能意味着后续缺乏新催化剂。历史上，利好消息被反复炒作后，往往迎来"利好兑现"式调整。

本次决策规避：不追高已充分定价的"确定性机会"。

教训3："错失恐惧"是亏损之源

激进分析师说"最大的风险是错过行情"，这话在牛市顶峰说过无数次，每次都导致惨重损失。

本次决策规避：坚持风险收益比原则，不因FOMO（错失恐惧）而偏离策略。

四、最终投资建议

核心裁决：持有（HOLD），偏向保守

  2. 基本面存在结构性疑问：净利率1.31%需要验证，而非假设改善

  3. 铜价高位风险不可忽视：当前铜价已在历史区间上沿，向下弹性大于向上

真实总账户净值 = 真实账户现金 + 全部真实持仓市值。这是执行层真正发生买卖和记账的总账户口径。
真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。

策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。

未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
