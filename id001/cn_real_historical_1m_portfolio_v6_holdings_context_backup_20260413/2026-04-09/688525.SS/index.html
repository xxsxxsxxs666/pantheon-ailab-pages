  2. 103亿是刚性支出，不是收入保证：即使晶圆价格锁定在低成本，但如果下游需求跟不上，这103亿晶圆就会变成堆积的库存和巨大的减值风险。平均每月近4.3亿的刚性支出，在需求波动面前是巨大的财务压力。

  3. “上游愿意签长单证明地位被认可”的逻辑谬误：换个角度想——为什么上游原厂愿意签24个月锁价合同？正是因为他们也看好未来价格走势，想提前锁定买家。这是双向博弈，激进分析师只看到一面。

四、保守分析师的正确判断与不足

保守分析师的核心贡献：

  1. 准确识别了采购合同的本质：刚性支出义务，而非利好来源
  2. 正确评估了禁令风险：发明专利诉讼远比5000万赔偿金额可怕
  3. 揭示了资金面隐忧：授信增加与合同签订的时间节点关联性
  4. 戳穿了市场情绪论调的虚假性：8.3%的弱势反弹是挣扎信号而非强势反转

保守分析师的不足：

  • 对诉讼直接影响评估略显过度悲观：虽然禁令风险真实存在，但5000万索赔金额确实相对有限，激进分析师的206:1对比有一定参考价值
  • 未能提供更精细的仓位管理建议：建议“减仓到30%以下”过于笼统，缺乏具体操作路径

“真实的不确定性在于：这是一个尾部风险不对称的交易。向下可能是30%-50%的跌幅，向上可能是20%-30%的涨幅。这个风险收益比，其实对激进分析师不利。”

持仓金额曲线 = 日末持股数 × 当天股价。
真实总账户净值 = 真实账户现金 + 全部真实持仓市值。这是执行层真正发生买卖和记账的总账户口径。
真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。

策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
