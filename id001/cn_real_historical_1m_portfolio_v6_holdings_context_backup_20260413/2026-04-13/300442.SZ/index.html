六、从辩论中得出的核心教训

关于激进分析师的反思

激进分析师最大的问题在于把"行业有前景"等同于"股票值得买"。他引用了云计算市场阿里云、华为云、腾讯云共存的例子，但他忽略了一个关键问题：这三家公司都有自己清晰的竞争优势和市场份额数据。润泽科技呢？它的竞争优势是什么？市场份额是多少？他一个都没有回答。

经验教训：投资个股时，必须问"为什么是这家公司而不是竞争对手"，而不是简单地说"赛道好所以公司好"。

关于保守分析师的反思

保守分析师提出的"等待完美买点"策略听起来理性，但实际上存在一个致命缺陷：如果市场不给你完美买点呢？ 如果润泽科技在84-85元获得了支撑并开始新一轮上涨，他的等待就变成了踏空。

更关键的是，他对"数据不可得"的处理方式存在方法论错误。基本分析报告明确说明了这是因为"历史截面数据"而非"数据缺失"，他将二者混为一谈。

经验教训：在信息不完美的市场中，需要区分"我们不知道"和"事实是负面的"，而不是简单地因为"不知道"就放弃判断。

关于中性分析师的反思

中性分析师指出了激进分析师"嘴上讲趋势交易，实际做追高"的矛盾，这是整场辩论中最犀利的观察。但他最终的建议"可以持有但不要重仓追高"却模糊不清——在89.18元，"持有"和"不追高"到底是什么意思？

经验教训：无论分析多么平衡，最终必须给出明确的可执行建议。投资者需要的是"做什么、什么价格做、做到什么程度"的三有答案，而不是"既可以这样也可以那样"的废话。

股价曲线 = 当天用于回放估值/执行的价格口径；当前日级回放默认使用 14:55 附近的可见价格或当日可用标记价格。
持仓金额曲线 = 日末持股数 × 当天股价。
真实总账户净值 = 真实账户现金 + 全部真实持仓市值。这是执行层真正发生买卖和记账的总账户口径。
真实总账户收益率 = 真实总账户净值 / 实验起始总账户净值 - 1，用来回答整个真实股票账户从实验开始到现在到底赚了还是亏了多少。它是累计收益率，不是相对前一天的日涨跌幅，所以即使某一天净值比前一天回落，只要仍高于实验起点，累计收益率依然可以是正数。
真实总账户现金曲线 = 真实账户里当前可用现金；真实总账户持仓市值曲线 = 真实总账户净值 - 真实总账户现金。
累计分配资金 = 首日持仓起点 + 后续从原始闲置现金拨给该股票的新增资金。这里只统计新增拨入，不把卖出回笼现金重复计入。
策略覆盖资产 = 策略覆盖现金 + 策略覆盖股票持仓市值。这里的“策略覆盖现金”只是展示层为了拆解这只股票策略效果而做的记账口径，不是执行层真的新开了一个现金账户。
策略拆分盈亏 = 策略覆盖资产 - 累计分配资金。
策略拆分收益率 = 策略拆分盈亏 / 累计分配资金。
已实现盈亏 = 已卖出部分的卖出收入 - 该部分对应成本。
未实现盈亏 = 当前剩余持仓市值 - 当前剩余持仓成本。
历史总盈亏 = 已实现盈亏 + 未实现盈亏，用来反映历史持仓成本视角，不是本轮实验的主收益口径。
